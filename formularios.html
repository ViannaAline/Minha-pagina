Estrutura de criação de formulários

<FORM ACTION="URL"> Os dados serão enviados ao endereço de e-mail especificado. </FORM>
<FORM ACTION="processa.php"> Serão processados por meio de um código PHP.
<FORM ACTION="URL" METHOD="GET" ou "POST"> GET: Utilizado para respostas únicas. POST: Utilizado para a transmissão de uma maior quantidade de dados.
<FACE="VERDANA">Nome:<INPUT TYPE=TEXT NAME="NOME" SIZE="20" MAXLENGHT="50"></p<> Cria um formulário para preenchimento do nome.
<FACE="VERDANA">Estado:<INPUT TYPE=TEXT NAME="UF" SIZE="2" VALUE="SP"> Cria um formulário para preenchimento do estado.
<P>Número do cartão de crédito:<INPUT TYPE=TEXT NAME="num_cartao" SIZE="16"></p>
<p>Senha: <INPUT TYPE=PASSWORD NAME="SENHA" SIZE="6"></p>
<INPUT TYPE=radio NAME="selecao-sexo"CHECKED>Feminino
<INPUT TYPE=radio NAME="selecao-sexo"CHECKED>Masculino Acrescenta uma caixa de opções de múltipla escolha.
<INPUT TYPE=checkbox NAME="HORÁRIO1" SIZE="1">13H ÀS 15H Acrescenta uma caixa de múltipla escolha.
<p>Referência:<TEXTAREA NAME="REF" ROWS="2" COLS="40"> Acrescenta uma caixa para digitar um texto. </TEXTAREA></P> <INPUT TYPE=SUBMIT VALUE="Enviar"> Acrescenta um botão para envio.
<INPUT TYPE=RESET VALUE="LIMPAR"> Acrescenta um botão para apagar informações inseridas.
<INPUT TYPE=HIDDEN NAME="CPF" VALUE="verdadeiro"> Adiciona um campo oculto e envia uma palavra chave, validação para o script CGI.
<SELECT NAME="cidades" SIZE="4"><OPTION>LONDRES<OPTION>PARIS</SELECT> Adiciona uma lista de opções.
<SELECT NAME="cidades"><OPTION>LONDRES<OPTION>PARIS</SELECT> Adiciona uma lista de opções abrindo uma caixa para escolha.
